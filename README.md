# project_homework

## Описание  
Репозиторий содержит проект «домашняя работа» — учебный проект на Java 8 с использованием Spring, FreeMarker и типовой структуры Maven.  
Цель проекта: показать подходы к организации кода, настройке CI/CD (наличие файла `.gitlab-ci.yml`), работе с шаблонами и простым веб-или бизнес-приложением.

## Содержимое  
- `.idea/` — конфигурационные файлы среды разработки IntelliJ IDEA.  
- `src/` — исходный код проекта на Java + шаблоны FreeMarker.  
- `pom.xml` — файл сборки Maven.  
- `.gitignore` — список игнорируемых файлов.  
- `.gitlab-ci.yml` — пример настройки конвейера CI/CD.  
- `README.md` — этот файл.

## Требования  
Для сборки и запуска проекта вам потребуется:  
- Java 8 (или более поздняя версия)  
- Maven 3.x
- postgresql
- Git для клонирования репозитория  
- (Опционально) IntelliJ IDEA или другая IDE для разработки.

## Установка и запуск  
1. Клонируйте репозиторий:  
   ```bash
   git clone https://github.com/DmitryKhm/project_homework.git
   cd project_homework

2. Сборка проекта с помощью Maven:
   mvn clean install

3. Запуск приложения
   mvn spring-boot:run
